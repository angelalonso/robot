# Information about the Arduino part
## Arduino clone "ISCP" Map - yeah I know, freaking clones' typos
```
-----------
  |1|3|5|  
  |2|4|6|  
  
1 Vcc  
2 MISO  
3 ICSP MOSI  
4 ICSP SCK  
5 ICSP GND  
6 ICSP RESET  
```

### Connection to Raspberry pi 1 B+ 
```
ISCP Pin NNumber -> Raspberry Pi Pin Number
               1 -> 2  
               2 -> 21  
               3 -> 19  
               4 -> 23  
               5 -> 6  
               6 -> 7  
```
